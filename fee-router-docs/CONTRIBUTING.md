# CONTRIBUTING.md

Köszönjük, hogy hozzájárulnál a Fee Router projekthez!

## Hogyan kezdj?
1. Forkold a repót, készíts branch-et: `feat/my-change`
2. Telepítsd a dev függőségeket, futtasd a lint/tesztek parancsot (ha van CI)
3. Küldj Pull Requestet részletes leírással

## PR elvárások
- Leírás, motiváció, breaking changes jelzése
- Ha publik API változik → dokumentáció frissítése
